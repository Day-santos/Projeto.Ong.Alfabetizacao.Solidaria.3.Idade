## 📘 Sobre o Projeto

A **ONG Alfabetização Solidária — 3ª Idade** é um projeto acadêmico desenvolvido na disciplina de **Desenvolvimento Front-End para Web**, do curso de **Análise e Desenvolvimento de Sistemas (ADS)**.  
O objetivo foi aplicar, de forma prática e evolutiva, os conhecimentos adquiridos em **HTML5**, **CSS3** e **JavaScript**, simulando o desenvolvimento real de uma aplicação web para uma organização do terceiro setor.

O site oferece uma experiência completa, incluindo:
- Informações institucionais da ONG;  
- Divulgação de **projetos sociais**;  
- Formulário de **cadastro de voluntários**;  
- **Modo noturno acessível** para inclusão visual;  
- Validação de formulários e feedback visual interativo.
